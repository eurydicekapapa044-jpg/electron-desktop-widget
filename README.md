# Pixel Desktop Widget (Electron)

![Banner](assets/banner.png)

A floating desktop widget app built with Electron featuring a pixel-style calendar and a local music player.  
Designed to behave like a lightweight always-on-top desktop widget similar to Rainmeter or macOS widgets.

---

## Features

- Live calendar (day and month)
- Local music player (MP3 support)
- Frameless floating widget window
- Always-on-top desktop mode
- Optional click-through mode
- Instant startup feel
- Pixel UI aesthetic
- Offline local file support
